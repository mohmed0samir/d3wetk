# Audio

The invitation plays Surat Ar-Rum, verse 21 ("ومن آياته أن خلق لكم من أنفسكم أزواجًا") after the envelope is opened. Its source is configured in the `wedding.music` value in `script.js`.

For a completely offline deployment, save a permitted recitation as `assets/music/rum-21.mp3` and change `wedding.music` to that local path.
